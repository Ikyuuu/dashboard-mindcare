## Cara Menjalankan Project

Ikuti langkah berikut untuk menjalankan project secara lokal:

### 1. Ekstrak File Project

Ekstrak file project terlebih dahulu.

### 2. Masuk ke Folder Project

Buka **Terminal / Command Prompt**, lalu masuk ke folder project:

```bash
cd path/folder/mindcare-dashboard
```

### 3. Install Dependencies

Install semua library yang dibutuhkan dengan menjalankan perintah berikut:

```bash
pip install -r requirements.txt
```

### 4. Jalankan Dashboard

Jalankan aplikasi menggunakan Streamlit:

```bash
streamlit run dashboard.py
```

### 5. Akses Dashboard

Jika dashboard tidak terbuka otomatis di browser, buka link berikut secara manual:

```text
http://localhost:8501
```
