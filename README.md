Cara Menjalankan Project

1. Ekstrak File Project
2. Masuk ke folder project, Lalu buka Terminal/Command Prompt
   cd path/folder/mindcare-dashboard
3. Install Dependencies
   pip install -r requirements.txt
4. Jalankan Dashboard
   streamlit run dashboard.py
5. Jika dashboard tidak terbuka, akses link localhost
   http://localhost:8501
